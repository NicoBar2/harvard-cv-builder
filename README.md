# 🎓 CVListo - Plantilla Harvard

[![GitHub Pages](https://img.shields.io/badge/deploy-GitHub%20Pages-brightgreen)](https://NicoBar2.github.io/harvard-cv-builder/)
[![React](https://img.shields.io/badge/built%20with-React-61DAFB)](https://reactjs.org/)
[![Tailwind CSS](https://img.shields.io/badge/styled%20with-Tailwind%20CSS-38B2AC)](https://tailwindcss.com/)

**CVListo** es una herramienta web moderna y gratuita diseñada para ayudar a estudiantes y profesionales a crear currículums de alto impacto siguiendo el estándar académico de **Harvard**.

🚀 **¡Pruébalo ahora!**: [https://NicoBar2.github.io/harvard-cv-builder/](https://NicoBar2.github.io/harvard-cv-builder/)

---

## ✨ Características Principales

- 📜 **Plantilla Harvard Oficial**: Diseño minimalista, elegante y optimizado para sistemas ATS.
- 🌓 **Modo Oscuro/Claro**: Interfaz adaptable para trabajar cómodamente en cualquier entorno.
- 📝 **Edición en Tiempo Real**: Visualiza tus cambios instantáneamente mientras escribes.
- 💾 **Persistencia Local**: Tus datos se guardan automáticamente en el navegador.
- 📥 **Exportación Versátil**:
  - **PDF de Alta Calidad**: Generado nativamente para máxima nitidez de texto.
  - **Word (.docx)**: Documento totalmente editable con márgenes y estilos preconfigurados.
- 🛠️ **Secciones Flexibles**: Educación, Experiencia, Proyectos, Certificaciones, Voluntariado y Publicaciones.

## 🛠️ Tecnologías Utilizadas

- **React 19** & **TypeScript**
- **Tailwind CSS** (Diseño responsivo y moderno)
- **Lucide React** (Iconografía)
- **docx** (Generación de archivos Word)
- **html2canvas** & **jsPDF** (Captura y procesamiento)

## 🚀 Instalación Local

Si deseas ejecutar este proyecto en tu propia máquina:

1. Clona el repositorio:
   ```bash
   git clone https://github.com/NicoBar2/harvard-cv-builder.git
   ```
2. Entra en la carpeta:
   ```bash
   cd harvard-cv-builder
   ```
3. Instala las dependencias:
   ```bash
   npm install
   ```
4. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
